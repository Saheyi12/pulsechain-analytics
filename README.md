## ✨ Features

- 🤖 **AI Price Predictions** - 7-day forecasts with confidence scores
- 📊 **Real-Time Market Data** - 10,000+ cryptocurrencies tracked
- 💼 **Portfolio Tracker** - Real-time P&L and asset allocation
- 🔔 **Smart Alerts** - Price, volume, and whale movement notifications
- 📝 **Auto Blog Generation** - AI-generated daily market analysis
- 💰 **Affiliate Program** - Earn commissions from referrals
- 🎨 **Premium Design** - Glass morphism, gradients, animations
- 🔒 **Security First** - CSP, HSTS, rate limiting, JWT auth

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/yourusername/pulsechain-analytics.git
cd pulsechain-analytics

# Install dependencies
npm install

# Set up environment
cp .env.example .env.local

# Set up database
npx prisma db push
npx prisma db seed

# Start development
npm run dev
